## Params
    
<ParamField path="key" type="string" required={true}>
  A unique string. Please see the [Keys and Resumability](/guides/resumability)
  doc for more info.
</ParamField><ParamField path="params" type="object" required={true}>
   Get Comments Input
<Expandable title="properties" defaultOpen={true}><ParamField path="block_id" type="string" required={true}>
   ID of the block
</ParamField><ParamField path="Notion-Version" type="string" required={true}>

</ParamField><ParamField path="start_cursor" type="string" required={false}>
   The cursor to start from. If not provided, the default is to start from the beginning of the list.
</ParamField><ParamField path="page_size" type="integer" required={false}>
   The number of results to return. The maximum is 100.
</ParamField></Expandable></ParamField>
